Password Strength Checker

A simple Python based password strength checker that analyzes passwords and tells how strong or weak they are.

Built to understand basic password security concepts like:

Length of password
Uppercase and lowercase letters
Numbers
Special characters
Common weak patterns

How It Works

The program checks:

Password length
Presence of uppercase letters
Presence of lowercase letters
Digits
Symbols

Based on these checks it calculates the overall strength.

Installation

Clone the repository

git clone https://github.com/heyiamrachappa/Password-Strength-Checker.git

Move into the project directory

cd Password-Strength-Checker

Run the program

python3 main.py
